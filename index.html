QuickPoses Redesign

redesigning the customization page on figure drawing reference website

Quickposes is a website that provides free images for artists to practice their figure drawing skills. Users can customize their practice sessions by selecting the subject they would like to draw, practice duration, and number of images. As a big fan of QuickPoses ‘ image library, I decided to redesign the most commonly used "Timed Practice" page to make it easier to navigate.

Note: This page currently displays a course project. Since I am not completely satisfied with how it turned out, I am currently in the process of revising this project with more user research and competitor analysis!

  • Project Type: individual project
  • Time: ongoing
  • Role: researcher, designer

identifying problems

Introduction contains some unnecessary information that can discourage the users from reading it. The download app ad has brighter colors than everything else on the page. It is also at a central location that disrupts the user flow. The menu bar only uses bolding to indicate selection, making it difficult to see what section the user is on at a glance. The drawing session customization section is clustered with lots of options It is unclear what the “procedure” section falls under. The information starting from the “Procedure” section seems to be drawing tips for the user, but the user has to read and figure it out themselves. Procedures and example videos can be redundant information for frequent users

acessibility evaluation

Assessing the accessibility of the Quickposes website with the help of WAVE Web Accessibility Evaluation Tool

The orange text provides very low contrast from the background Alternative text missing for various images The “custom intervals” note is too small and difficult to read

low fidelity wireframing

With the usability problems and accessibility issues in mind, I created wireframes for the redesign

  1. Unify the fonts used in the page
  2. Edit the introduction, so it only contains the most basic information about the website
  3. Move the “get the app” advertisement to the navigation bar. Even though the advertisments are not visual pleasing, I don't want to simply remove them, since they are necessary to support the website
  4. Change the styling of image categories to look less like buttons
  5. Use shading or color to better indicate selection on the menu bar
  6. Add “image direction” as a drawing session customization option, so users can filter horizontal or vertical images that make the most use out of their screen spaces
  7. Move the Quickpose certificate option to the center of the screen to separate it from the customization section
  8. Move the “start” button to the center of the screen to separate it from the customization section and to provide better flow from the previous certificate section
  9. Move “Tips and Tricks” to a separate page, so frequent users do not need to see the same information every time
  10. Move the “About” section to a separate page to declutter the current page
